Quantum Calculus

This note is for $q$-calculus. One needs to keep in mind that my motivation to do quantum calculus is not for difference equations but for quantum groups, so $ q = e^{\frac{\hbar}{2}}$. One can view $q$ as a generic nonzero complex number. But we are also interested in cases when $q$ is a root of $1$.

Basic definitions

We begin with $q$-integers. For $ n \geqslant 0 $,

$$ \tag{I1} [n]_q := \frac{ q^n - q^{ -n } }{ q - q^{ -1 } } = q^{ -n + 1 } + q^{ -n + 3 } + \cdots + q^{ n - 3 } + q^{ n - 1 }. $$

For simplicity, we define $[-n]_q = -[n]_q$. Then $[1]_q = 1,[0]_q = 0$. The summation formula is

$$ \tag{I2} [n+m]_q = q^{m}[n]_q + q^{-n}[m]_q = q^{-m}[n]_q + q^{n}[m]_q. $$

Using $ \mathrm{(I2)} $, we can write down the product formula, for $m \geqslant n \geqslant 0$,

$$ \begin{align}
[m]_q [n]_q & = [m]_q ( q^{- n + 1} + q^{ - n + 3 } + \cdots q^{n - 3} + q^{ n - 1 } ) \\ & = ( [m + n - 1]_q - q^m[n - 1]_q ) + ( [m + n - 3]_q - q^m [n - 3]_q ) + \cdots \\ & \quad + ( [m - n + 1]_q - q^{m} [1 - n]_q ) \\ & = [ m + n - 1 ]_q + [ m + n - 3 ]_q + \cdots [ m - n + 1 ]_q. \tag{I3}
\end{align} $$

We will also define $q$-factorials and $q$-binomial coefficients as

$$ [n]_q ! = \prod \limits _{k=1} ^n [n]_q, \quad \left [ \begin{matrix} m \\ n \end{matrix} \right ]_q := \frac{[m]_q!}{[n]_q![m-n]_q!}, \tag{I4} $$

where $m \geqslant n \geqslant 0$. Using $ \mathrm{(I2)} $ the recursion relation of $q$-binomial coefficients is given by

$$
\left [ \begin{matrix} m+1 \\ n+1 \end{matrix} \right ]_q = q^{ (n-m) } \left [ \begin{matrix} m \\ n \end{matrix} \right ]_q + q^{ (n+1) } \left [ \begin{matrix} m \\ n+1 \end{matrix} \right ]_q. \tag{I5}
$$

That is all for $q$-numbers. We will then work with functions[1], i.e., $q$-polynomials or $q$-series, which are usually generating functions of sequences of $q$-numbers. We will also consider $q$-analogue of differential operators, which are usually difference operators.

Def I.1. Given any function $ f(x) $ of one variable $ x $, its $q$-differential is
$$
\partial_q f(x) = \frac{f(qx) - f(q^{-1}x)}{qx - q^{-1}x}. \tag{I6}
$$

Def I.1 makes sense for we have $ \partial_q x^n = [n]_q x^{n-1} $ and we have the following Leibniz Rule,
$$ \tag{I7}
\partial_q fg(x) = g(qx)\partial_q f(x) + f(q^{-1}x) \partial_q g(x).
$$

Remark One might have noted that in my setup, $q$ and $q^{-1}$ can be exchanged with each other.

The $q$-analogue of $(x+a)^n$ is the polynomial

$$ \tag{I8}
(x+a)_q^n := \Big \{ \begin{array}{ll} 1 & \text{if } n= 0, \\ (x+aq^{-n+1})(x + aq^{-n + 3}) \cdots (x + aq^{n-1}) & \text{if } n\geqslant 1. \end{array}
$$

Directly by $\mathrm{(I6)}$,
$$
\partial_q (x+a)^n_q = [n]_q(x+a)_q^{n-1}. \tag{I9}
$$

The product formula for $(x+a)^n_q$ is twisted too,

$$
( x + a )^{ m + n }_q = ( x + q^{-n} a )^m_q ( x + q^m a)^n_q, \tag{I10}
$$

and in fact, we have the following binomial formula (just as you may have guessed, $(x+a)^n_q$ is the generating function of $q$-binomial coefficients),

$$ \tag{I11}
(x+a)^n_q = \sum\limits_{j=0}^n \begin{bmatrix} n \\ j \end{bmatrix}_q a^{n-j} x^j.
$$

One can prove this via $\mathrm{(I9)}$ or using $q$-Taylor expansion we will talk about later. There is a crucial point about $(x+a)^n_q$, if $n$ is even,
$$
(x-a)_q^n \Big | _{x = a} \neq 0.
$$

In fact, we can introduce the very famous $q$-Pochhammer symbol,

$$ \tag{I12}
(a;q)_n := (1-a)(1-aq) \cdots ( 1-aq^{n-1} ).
$$

$q$-Pochhammer symbol is also called $q$-shifted factorial by many people (We shall not use this name, for we already have our version of $q$-factorials.) One can easily write down the relations between our notations and $q$-Pochhammer symbol,

$$ \tag{I13}
[n]_q! = \frac{ -q^{ -\frac{ n(n+1) } {2} } } { ( q^{-1} - q )^n }( q^2 ; q^2 )_n, \quad (x - a)^n_q = x^n ( \frac{a q^{ - n + 1} }{x} ; q^2 )_n.
$$

Hence,

$$
(x-a)_q^n \Big | _{x = a} = a^n ( q^{ - n + 1 } ; q^2 )_n .
$$


Thm I.2. For any polynomial $f(x)$ and any number $a$, we have the following $q$-Taylor expansion,
$$ \tag{I14}
f(x) = f(a) + \sum\limits_{n=1}^\infty ( \partial_q^n f)(a) \frac{ (x - a)^n_q - a^n ( q^{ - n + 1} ; q^2 )_n }{ [ n ]_q ! }.
$$

Proof. By induction on the degree of $f$. Assume $\mathrm{(I14)}$ holds for all $f$ with $\deg f \le n$, then for $f$ with $\deg f = n$, using the fact that $(x - a)^k_q$ is with degree $k$ and monic,
$$
f(x) = \sum\limits_{k=0}^n c_k (x - a)^k_q,
$$
with $c_k \in \mathbb{C}$. By taking $\partial_q$, we have for $k \geqslant 1$
$$
c_k [k]_q = (\partial_q^k f)(a) \frac{1}{ [ k - 1 ]_q! }.
$$
In other words, for $k \geqslant 1$,
$$
c_k = (\partial_q^k f)(a) \frac{1}{ [ k ]_q! }.
$$

Then the result follows by comparing the constant terms. $\blacksquare$
