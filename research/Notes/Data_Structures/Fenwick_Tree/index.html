Binary Indexed Tree

A binary indexed tree or Fenwick tree $BIT = {BIT_0,BIT_1,\cdots, BIT_n}$ is designed for get prefix sums of an array ${a_1,a_2,\cdots,a_n}$, whose items are allowed to be mutated.

The tree structure of $BIT$ is given as follows:

  • Each node is an item in $BIT$ and carries the same index from $BIT$
  • The root node is $BIT_0 = 0$
  • A child node $BIT_k$ has Hamming distance $1$ away from their parent node $BIT_{P(k)}$.
  • The depth of a node indexed by $ind$ equals to the Hamming distance of $ind$

$$
\sum_{k=1}^\ell a_k = BIT_\ell + BIT_{P(\ell)} + BIT_{P^2(\ell)} + \cdots + BIT_{0}.
